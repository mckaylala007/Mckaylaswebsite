# McKayla Website

## Run locally
1. Install Node.js 18+
2. Run `npm install`
3. Run `npm run dev`

## Deploy
Import this repo into Vercel, then connect the domain `mckaylamaroney.com`.

## Notes
- Replace placeholder Amazon links before launch.
- Replace public image URLs with approved brand photos.
- Wire the email form to Klaviyo/ConvertKit.
- After domain setup, remove `parent=localhost` from the Twitch embed if desired.
